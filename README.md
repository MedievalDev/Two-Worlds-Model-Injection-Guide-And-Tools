# Two-Worlds-Model-Injection-Guide-And-Tools
Complete guide and tools for importing custom 3D models into Two Worlds 1 or editing existing ones. Covers VDF format, Blender workflow, material/shader setup, NTF node structure, and editor integration. Includes VDF-to-OBJ converter, OBJ-to-VDF converter, Blender plugin, and NTF Editor.
